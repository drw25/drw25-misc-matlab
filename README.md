# drw25-misc-matlab
-- miscellanous MATLAB tools

FVtoPOVRay converts a MATLAB faces and vertices structure (eg. as made by the isosurface function) to an input file for the POV-Ray raytracer (http://www.povray.org)

FVtoVRML converts a MATLAB faces and vertices structure (eg. as made by the isosurface function) to VRML

LUT2Map converts ImageJ colour lookup tables to MATLAB's colormap format

SGolayND calculates Savitzky-Golay convolution coefficients (https://en.wikipedia.org/wiki/Savitzky%E2%80%93Golay_filter) to smooth a discrete evenly-spaced dataset and estimate its derivatives, in an arbitrary number of dimensions

txtwaitbar is a text-based progress bar, suitable for use in parfor loops
